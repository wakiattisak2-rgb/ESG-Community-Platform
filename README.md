# ESG Community Platform (Aetros)

## Live website (ใช้ลิงก์นี้ — ไม่ใช่หน้า Code บน GitHub)

**https://aetros-esg.vercel.app**

## Local development

```powershell
cd aetros
npm install
npm run dev
```

เปิด **http://localhost:5173** — อย่าดับเบิลคลิก `index.html` ใน Explorer (จะเห็นโค้ด/หน้าว่าง)

## Project map

| ที่ | คืออะไร |
